# 🎯 فكرة المشروع

يقوم المستخدم برفع صورة لورقة نبات مصابة، ثم يقوم النظام بـ:

1. معالجة الصورة باستخدام نموذج ذكاء اصطناعي
2. تحليل المرض المتوقع
3. حساب نسبة الثقة
4. جلب معلومات المرض من قاعدة معرفة محلية
5. عرض:

* اسم المرض
* نسبة الثقة
* الأعراض
* العلاج
* الوقاية
* المبيدات المتوفرة محلياً
* موسم الخطورة
* نصائح زراعية

كما يحتوي النظام على:

🌿 مساعد زراعي ذكي (AI Chatbot)
للإجابة عن الأسئلة الزراعية باللغة العربية.

# ⚙️ كيف يعمل النظام

## 1) رفع الصورة

يقوم المستخدم برفع صورة للنبات المصاب.

---

## 2) معالجة الصورة

يتم:

* تحويل الصورة إلى RGB
* إعادة تحجيمها إلى:

```python
224x224
```

* تجهيزها للنموذج

---

## 3) التنبؤ بالمرض

يقوم النموذج بإرجاع:

* المرض المتوقع
* نسبة الثقة

---

## 4) التحقق من الثقة

إذا كانت نسبة الثقة منخفضة:

```python
if confidence < 0.60
```

يتم رفض النتيجة لتقليل التشخيصات العشوائية.

---

## 5) جلب المعلومات الزراعية

يتم تحميل معلومات المرض من ملفات JSON محلية داخل:

```text
knowledge_base/disease_json
```

---

# 🗂️ قاعدة المعرفة الزراعية

كل مرض عبارة عن ملف JSON مستقل يحتوي على:

* الاسم العربي
* الأعراض
* العلاج
* الوقاية
* المبيدات المحلية
* موسم الخطورة
* نصائح زراعية

مثال:

```json
{
  "disease": "Apple Scab",
  "arabic_name": "جرب التفاح",
  "symptoms": [
    "بقع زيتونية بنية على الأوراق"
  ],
  "treatments": [
    "رش مبيدات فطرية"
  ]
}
```
